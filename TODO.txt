1. Маленькие поправки к Дизайну

	1.1. Разбить стаф на графы:
		- File:
			+ Open
			+ Save
			+ Export Midi
			+ Export Png	
		- Navigation:
			+ Play/Stop
			+ Left
			+ Right
			+ Up
			+ Down
			+ To Begin
			+ To End

	1.2. Вынести в StaffConfig:
		+ Settings
		+ Change Playback Device
		+ Disable Midi Input
		+ Enable Midi Input
